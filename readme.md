# Symfony Initiation

Récupérer le projet
```
git clone https://github.com/olivierPoussel/symfony-initiation.git
```

Installer le projet
```
composer install
```

Lancer le projet
```bash
symfony server:start
# ou
composer require symfony/web-server-bundle
php bin/console server:start
# ou
php -S localhost:8000 
```
